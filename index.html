Metode Koneksi

Pilih cara menghubungkan bot dengan nomor WhatsApp

🔀
QR Code + Pairing Code
Bot generate QR dan pairing code sekaligus. Kamu bisa pilih mau pakai yang mana saat pertama run. Paling fleksibel.
✦ Rekomendasi
📷
QR Code Saja
Scan QR code yang muncul di console Pterodactyl dengan WhatsApp. Buka WA → Perangkat Tertaut → Tautkan Perangkat → Scan.
🔑
Pairing Code Saja
Bot tampilkan kode 8 digit di console. Buka WA → Perangkat Tertaut → Tautkan Perangkat dengan Nomor → Masukkan kode.
📋 Info Session
Session disimpan otomatis di folder ./session/ di server Pterodactyl.
Jika bot logout atau perlu scan ulang → hapus folder session/ dan restart server.
Jangan hapus folder session jika bot masih berjalan normal!
